# 📈 Stock Analysis: How Should I Invest £1000?

## Project Overview

This project investigates how a beginner investor could allocate a £1000 portfolio between ETFs and individual stocks using historical market data.

Using Python and financial data from Yahoo Finance, the project compares the performance and risk characteristics of different investment choices over the past five years.

---

## Objectives

- Collect historical stock price data automatically
- Clean and preprocess financial datasets
- Compare investment performance
- Measure investment risk
- Recommend a simple diversified portfolio

---

## Assets

- Vanguard FTSE All-World UCITS ETF (VUAG.L)
- iShares NASDAQ 100 UCITS ETF (EQGB.L)
- Microsoft (MSFT)
- NVIDIA (NVDA)
- Alphabet (GOOGL)

---

## Analysis Performed

### Data Collection

Downloaded five years of historical prices from Yahoo Finance.

### Data Cleaning

- Checked missing values
- Removed incomplete observations

### Performance Analysis

Calculated

- Total Return
- CAGR (Annualised Return)

### Risk Analysis

Calculated

- Annual Volatility
- Maximum Drawdown
- Sharpe Ratio

### Correlation Analysis

Compared the relationships between different assets to evaluate diversification.

---

## Conclusion

Historical results suggest:

- NVIDIA generated the highest return but also carried the highest volatility and drawdown.
- ETFs provided more stable long-term performance.
- Google offered a good balance between return and risk.

For a beginner investor, a diversified portfolio with approximately **70% ETFs and 30% individual stocks** provides a reasonable balance between growth and risk.
